📊 Сравнение компаний — все показатели сразу
На одной диаграмме видно, в чём каждая компания сильна, а в чём слаба. Каждая ось — отдельный показатель, нормированный в процентиль от остальных в этом же периоде (0 = худший, 100 = лучший). Для «меньше — лучше» метрик (ND/EBITDA, Долг) знак перевёрнут, так что на радаре всегда «чем дальше от центра, тем лучше».

⚡ Синхронизация между устройствами
Синхронизируется: портфель, watchlist, календарь, отчётности, эталоны сверки, AI-ключ. UI-настройки (видимость панелей, маска) остаются локально — зависят от размера экрана.
🔑 Код синхронизации — без GitHub, без регистрации, без аккаунтов. Данные шифруются AES-256-GCM у вас в браузере и кладутся анонимным blob'ом на jsonblob.com. Тот же код на другом устройстве расшифровывает и подгружает. Ключ шифрования — часть кода, сервер его не видит.
📡 ГИР БО — прокси для автоматической подтяжки 5 лет РСБУ по ИНН
ГИР БО (bo.nalog.gov.ru) не отдаёт CORS-заголовки браузеру, поэтому нужен прокси. По умолчанию — публичный corsproxy.io. Для приватности и надёжности можно развернуть свой Cloudflare Worker (5 минут, бесплатно): код в файле cf-worker.js в репозитории, инструкция в README. URL вашего Worker'а вписывайте сюда.
Соглашение: если URL заканчивается на ? или = — прокси получит target-URL после; если на / — добавится https://bo.nalog.gov.ru/...; иначе — путь api подставится напрямую.
💼 Офлайн-код — без сети и без сервера, подходит когда блокировщик / корпоративный firewall режет fetch
Собираем тот же снапшот (портфель, watchlist, reportsDB, эталоны, AI-ключ), сжимаем gzip'ом и кодируем в base64. Длинная текстовая строка копируется на другое устройство любым способом (Telegram Saved Messages, e-mail себе, AirDrop, заметки) и вставляется в это же поле. Никаких внешних серверов.
Альтернатива: через GitHub Gist (если у вас один и тот же GitHub-аккаунт на всех устройствах)
Данные хранятся в приватном Gist на вашем GitHub-аккаунте. Если у вас разные аккаунты на разных устройствах — используйте код синхронизации выше.
Создать: github.com → Settings → Developer settings → Personal access tokens → New token → выбрать gist → Generate
Скопируйте ID из URL: gist.github.com/ваш_id

📗 Импорт audit-it
audit-it.ru даёт до 11 лет РСБУ (2015-2025), включая ВДО и санкционные компании. Серверный парсинг блокируется anti-bot'ом, но твой браузер страницы открывает нормально. Два способа:
🔖 Способ 1: bookmarklet (1 клик после настройки)
  1. Нажми кнопку ниже → в буфер скопируется код закладки.
  2. В браузере: Ctrl+Shift+B (панель закладок) → правый клик на панели → «Добавить страницу» → Имя: «📗 audit-it → БондАналитик», URL: Ctrl+V (вставить из буфера) → Сохранить.
  3. Теперь на любой странице audit-it.ru/buh_otchet/… клик по этой закладке → JSON в буфере.
  4. Возвращайся сюда → «📋 Вставить из буфера» ниже.
📋 Способ 2: paste HTML (без настройки)
  1. На странице компании на audit-it нажми Ctrl+U (просмотр исходного кода).
  2. В открывшейся вкладке: Ctrl+A → Ctrl+C (скопировать всё).
  3. Вернись сюда → Ctrl+V в поле ниже → «🔍 Распознать».
