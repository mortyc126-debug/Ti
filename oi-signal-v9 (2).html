Почему нужен прокси?
T-Invest API блокирует запросы из браузера (CORS). Нужен бесплатный Cloudflare Worker — это займёт 2 минуты.

1. workers.cloudflare.com → Start for free → Create Worker → Edit code
2. Замени весь код на:
export default {
  async fetch(request) {
    if (request.method === 'OPTIONS')
      return new Response(null, { headers: cors() });
    const path = new URL(request.url).pathname + new URL(request.url).search;
    const target = 'https://invest-public-api.tinkoff.ru' + path;
    const body = request.method !== 'GET' ? await request.arrayBuffer() : undefined;
    const resp = await fetch(target, {
      method: request.method,
      headers: {
        'Authorization': request.headers.get('Authorization') || '',
        'Content-Type': request.headers.get('Content-Type') || 'application/json',
      },
      body,
    });
    return new Response(await resp.arrayBuffer(), {
      status: resp.status,
      headers: { ...cors(), 'Content-Type': resp.headers.get('Content-Type') || 'application/json' },
    });
  }
};
function cors() {
  return {
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Methods': 'GET, POST, OPTIONS',
    'Access-Control-Allow-Headers': 'Authorization, Content-Type',
  };
}
3. Save & Deploy → скопируй URL (вида https://xxxxxxx.workers.dev)
4. Вставь URL в поле "прокси" выше → Enter

Факт. движение след. дня:
Логика сигнала · данные за — → прогноз на —
Ядро: OI физиков + цена (4 квадранта)
Цена↑ + ОИ физ↑ → перегрев, толпа входит на хаях → медвежий (−1.5)
Цена↓ + ОИ физ↑ → добирают на падении, каскад продолжается → медвежий (−2.0)
Цена↑ + ОИ физ↓ → физики сдаются на росте → рост без перегруза → бычий (+1.5)
Цена↓ + ОИ физ↓ → ликвидации, топливо иссякает → разворот вероятен → бычий (+1.0)
Дополнительные факторы
Сила квадранта: нормированное Δ ОИ × нормированное Δ цены → интенсивность
Total ОИ (все стороны): растёт/падает → подтверждение/ослабление
3-дневный тренд квадрантов → устойчивость сигнала
МM-структура (юр.Ш >> юр.Л) → автоопределение, не влияет на ядро
MTF согласованность (свечи) → ±0.5
Память (вкл.) → поправка на истор. точность
