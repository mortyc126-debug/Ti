⚙ Настройки индикатора
Цена + фазы накопления (синий фон = флэт с имбалансом; пунктир = диапазон фазы; 🔺оранж. = тренд-истощение)
Сигнал дисбаланса / Потенциал выхода (красный = физики в лонг → ожидаем вниз; зелёный = в шорт → ожидаем вверх)
Стресс-риск / Накопленный выброс (красные столбики = цена за пределами опорного уровня; оранжевая линия = сила выброса)
Точность системы на истории
Нажми «Рассчитать» после загрузки данных.
Калибровка по пулу инструментов (каждый тикер — отдельный независимый инструмент, не роллируется вместе с остальными; проценты/z-score считаются по своей истории каждого — пул усиливает статистику, а не смешивает масштабы)
Нажми «Заполнить всеми из базы» (или впиши тикеры вручную), затем «Рассчитать по пулу».
