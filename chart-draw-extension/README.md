# Разметка графика (расширение для Тинькофф веб-терминала)

Рисование поверх веб-терминала — то, ради чего приходится уходить в TradingView:
уровни, линии, лучи, каналы/параллели, карандаш. Чтобы отметить уровень прямо в
терминале и не переключаться.

## Установка (Chrome/Edge, режим разработчика)

1. `chrome://extensions` → включить «Режим разработчика».
2. «Загрузить распакованное» → выбрать папку `chart-draw-extension/`.
3. Открыть веб-терминал (`tbank.ru` / `tinkoff.ru`) — справа появится панель «✎ Разметка».

## Инструменты

- **⭶ выбрать** — выделить фигуру, тащить её или её точки, менять цвет/толщину, удалять (Delete).
- **━ уровень** — горизонтальная «палка» на всю ширину + цветной ярлык (чётко видно). Клик по цене.
- **╱ линия** — трендлиния (2 точки, концы двигаются).
- **⟶ луч** — от точки, продлевается за край.
- **▱ канал** — базовая линия + параллель; ширину канала тянуть за среднюю точку (в режиме «выбрать»).
- **✎ каранд.** — свободная линия.
- Цвет и толщина — общие для новых фигур и применяются к выделенной.
- **✕ очистить** — стереть всю разметку на странице. **спрятать / Alt+D** — скрыть слой (терминал полностью кликабелен).

Разметка **сохраняется** (localStorage) по ключу страницы+тикера и переживает
перезагрузку.

## v1: важное ограничение — экранные координаты

Сейчас линии живут в **пиксельных координатах экрана**, НЕ привязаны к цене/времени
графика. Практический эффект: при прокрутке/масштабировании графика разметка не
едет вместе со свечами. Для быстрой отметки уровня «здесь и сейчас» это ок; для
стойких уровней нужна привязка к цене.

**Почему так:** привязка требует читать шкалу цены терминала (его canvas/оси) —
это делается по реальной странице терминала. Прислать бы фрагмент DOM графика
(или пару скриншотов с ценовой осью) — тогда добавлю режим «якорить к цене»:
уровень будет стоять на своей цене при любом скролле.

## Дорожная карта

1. **Привязка к цене** (якорь горизонтальных уровней к price-оси терминала) — приоритет.
2. Ввод подписи уровня (двойной клик по ярлыку) + значение цены в ярлыке.
3. Магнит к экстремумам/круглым уровням.
4. Второй слой — наложение расчётов «Сигнальных моделей» (из indlab) как индикаторной панели рядом.

## Тех. детали

- MV3, content-script `overlay.js` + `overlay.css`, без сети и без токена (чистое рисование).
- Тумблер слоя — **Alt+D**.
- API для отладки/тестов — `window.__chartDraw` (getShapes/setTool/addShape/save/load/clear).
