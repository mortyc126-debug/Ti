# РАЗДЕЛ 1: MARKET STRUCTURE

## BOS / CHoCH / HH-HL-LH-LL

### Базовые определения

**BOS (Break of Structure)** — цена пробивает предыдущий значимый хай/лой в направлении тренда. Подтверждение что тренд жив, не сигнал входа.

**CHoCH (Change of Character)** — первый пробой структуры ПРОТИВ тренда. Предупреждение о смене фазы, не разворот сам по себе.

**HH/HL** — восходящая структура (Higher High / Higher Low)
**LH/LL** — нисходящая структура (Lower High / Lower Low)
**HH+LL или LH+HL** — сломанная структура, переход между фазами

---

### Связь с 5 фазами системы

| Фаза | Структура | Что искать |
|---|---|---|
| Накопление | LH+HL внутри диапазона | Каждый HL выше предыдущего = скрытое давление вверх |
| Выброс | BOS вверх на объёме | Пробой последнего HH + Klinger переключение |
| Каскад | HH+HL монотонно | Каждый HL = точка входа на откате |
| Затухание | HH есть, HL начинает понижаться | LH вместо HH = первый CHoCH |
| Разворот | CHoCH + BOS вниз | Klinger пересекает ноль одновременно |

---

### Необычные интерпретации

**Глубина HL = агрессивность накопления**

Откат к HL на 30% от предыдущего импульса = сильная структура, крупняк держит.
Откат к HL на 60%+ = слабая структура, накопление ещё не завершено.
Чем мельче откаты → тем агрессивнее накопление.

**CHoCH ≠ разворот, CHoCH = предупреждение**

Первый CHoCH часто ловушка. Настоящий разворот = CHoCH **плюс BOS в противоположную сторону**. Между ними 2-5 свечей флага где:
- Fisher RSI всё ещё в крайности старого направления
- Klinger ещё не пересёк ноль
- Это и есть ловушка для тех кто входит на CHoCH

**Скорость формирования структуры = измеритель энергии**

- HH сформировался за 2 свечи после HL = быстрая структура, каскад здоров
- HH за 8 свечей после HL = медленная структура, энергия падает
- Расстояние между HL и следующим HH растёт (2→4→8 свечей) = затухание раньше чем покажут индикаторы

**Equal Highs как пул ликвидности**

Два HH почти одинаковые (разница < 0.2%) = там стопы лонгов. Рынок придёт туда:
- Пробой вверх и удержание = BOS настоящий
- Sweep через Equal Highs + возврат = охота за стопами перед падением

После sweep Equal Highs + CHoCH вниз = настоящий разворот, не ложный.

**Структура внутри структуры (фракталы)**

Дневная HH+HL (восходящая) + часовая LH+LL (нисходящая) = не противоречие.
Часовая LH+LL это откат внутри дневного HL.
Точка входа: когда часовая структура даёт CHoCH вверх = вход в дневной HL.

**Дивергенция на структурном уровне**

Цена делает HH (структурно выше) — Klinger делает LH (структурно ниже).
Это двойная дивергенция: на уровне цены И на уровне осциллятора. Сильнее чем просто «Klinger не подтвердил».

**HL как точка входа точнее BOS**

BOS — это уже +1-2% от HL. Реальная точка: войти на формировании HL когда откат остановился и появилась первая бычья свеча с объёмом. Подтверждение:
- Fisher RSI ещё в бычьей зоне
- Klinger не упал
- SAR не переключился

# РАЗДЕЛ 2: VOLUME PROFILE

## POC / VAH / VAL / HVN / LVN

### Структура профиля

```
VAH ──── верхняя граница 70% объёма
HVN ···· узлы высокого объёма (магниты, тормозители)
POC ════ максимальный объём (центр тяжести)
HVN ···· узлы высокого объёма
LVN ···· пустые зоны (ускорители)
VAL ──── нижняя граница 70% объёма
```

---

### POC

**POC как термометр уверенности рынка**

POC медленно ползёт вверх каждый день (0.1-0.2%) пока цена стоит = скрытое накопление. Видно раньше чем любой индикатор.

**POC vs цена в ATR**

```
< 0.5 ATR расстояния  = цена у баланса, боговик
1-2 ATR               = здоровый каскад, POC тянет назад как резинка
> 3 ATR               = экстремум, либо каскад заканчивается либо начинается новый режим
```

**Смена роли POC**

POC пробит снизу вверх + цена вернулась + POC держит как поддержку = структура здорова.
POC пробит обратно = весь профиль переворачивается, POC становится сопротивлением.
Это CHoCH на уровне объёма.

**Composite POC**

Месячный профиль = один POC. Разбить на недели = четыре POC. Когда два-три недельных POC кластеризуются в 0.5% диапазоне = кластер POC. Сильнее любого Фибо.

---

### VAH / VAL

**VAH/VAL как предсказатель скорости**

Цена внутри Value Area движется медленно (там есть и спрос и предложение).
Цена вне Value Area летит — там никого нет.
Это объясняет почему каскады выглядят как прыжки: они идут через LVN зоны между профилями.

**Открытие выше VAH — принятие или отторжение**

Сессия открылась выше VAH вчерашнего дня:
- Цена продолжает вверх (принятие нового уровня) = каскад продолжается
- Цена возвращается в Value Area за первые 30 минут (отторжение) = ложный пробой, разворот вниз

Автоматический фильтр BOS в реальном времени.

**VAL как стоп-уровень точнее ATR**

На откатах в каскаде вверх цена часто тормозит у VAL текущего дня. Стоп под VAL точнее чем «ATR × 1.5» — там реальный объём, не математика.

---

### HVN (High Volume Node)

**HVN как замедлитель каскада**

Цена встречает HVN → AC Oscillator падает именно здесь даже если ADX ещё высокий.
HVN = точка частичного выхода, не полного. Каскад продолжится после «переваривания».

**Количество HVN = сложность прохода**

- Один HVN = цена проходит за 1-2 свечи
- Три HVN подряд = консолидация 3-7 свечей
- Klinger держит экстремум пока цена «жуёт» HVN = каскад не закончился

**HVN как зона входа на откате**

После прохода через HVN цена часто возвращается к нему на первом откате.
Это и есть твой HL при восходящем каскаде.
Вход на откате к HVN + Klinger держит + Fisher держит = лучшая точка добавления.

**Старые HVN работают через недели**

HVN от месячного профиля работает как магнит даже через 3-4 недели.
Когда цена приближается — жди замедление даже если текущий дневной профиль его не показывает.

---

### LVN (Low Volume Node)

**LVN = ускоритель, не пустота**

Когда цена входит в LVN — нет ни продавцов ни покупателей. Цена летит до следующего HVN или VAH/VAL другого профиля. Именно через LVN происходят свечи на 3-5% за час.

**LVN между двумя профилями = воздушный карман**

Вчерашний VAH на 100, позавчерашний VAL на 103 = LVN 100-103. 3% без сопротивления.
BOS на уровне 100 = цель минимум 103 автоматически.

**LVN как измеритель потенциала**

Перед входом: сколько LVN до следующего HVN?
- Один LVN в 2% = минимальная цель +2%
- Два LVN суммарно 5% = цель +5%

**LVN после долгого боговика = взрывной потенциал**

Choppiness высокий 10 дней + VAH пробита + LVN 4% выше = накопленная энергия × пустое пространство. Объясняет движения 8-15% за 2-3 дня.

---

### Volume Profile + остальная система

| VP уровень | Комбинация | Вывод |
|---|---|---|
| POC далеко > 2 ATR | ADX высокий | Каскад здоров, резинка натянута |
| BOS через VAH | ADX+10 + BB Width×1.5 | Идеальный вход: структура + режим + пространство |
| Откат к HVN | Volume < импульса × 0.4 | Точка добавления |
| LVN выше VAH | Fisher переключился + AC растёт | Цель = следующий HVN, без сопротивления |
| POC поменял роль | CHoCH + Klinger ноль | Настоящий разворот |

# РАЗДЕЛ 3: WYCKOFF УГЛУБЛЁННО

## Spring / Upthrust / SOS / SOW

### Spring

**Глубина spring = сила каскада**

Spring 0.3% ниже лоя = слабый, мало стопов собрали.
Spring 1.5-2% = глубокий, много стопов = каскад пропорционально сильнее.
Глубина spring + длина LVN выше = формула потенциала движения.

**Соотношение объёмов**

```
Объём пробоя 500k, возврат 300k = нормально, крупняк вошёл при пробое
Объём пробоя 500k, возврат 600k = аномалия, ещё агрессивнее входят на возврате → резкий каскад
Объём пробоя 200k, возврат 400k = ложный spring, разворота не будет
```

**Тихий Spring (No Supply Test)**

Spring без пробоя лоя — цена просто касается его на минимальном объёме (в 3-5 раз ниже среднего) и отскакивает.
Fisher RSI не уходит в отрицательную зону несмотря на касание лоя.
Крупняк уже всё набрал, стопов практически нет = самое взрывное движение.
Сложнее увидеть, сильнее результат.

**Spring на разных уровнях**

- На лое текущего боковика = обычный spring
- На VAL volume profile = структурный spring (сильнее)
- На старом HVN который стал поддержкой = глубокий spring (самый сильный)

Чем «старше» уровень = тем больше стопов = тем сильнее spring.

---

### Upthrust

**Upthrust как финальная ловушка в затухании**

Признаки что это upthrust а не BOS:
- Пробой хая за одну свечу, потом немедленный возврат (не держит 2+ свечи)
- Объём огромный на пробое, но Fisher RSI уже не делает новый максимум
- Klinger дивергенция уже была до этого

Разница от BOS: BOS держит пробой 2+ свечи. Upthrust возвращается за одну.

**Upthrust + Equal Highs = двойная ловушка**

Sweep Equal Highs = стопы со старого хая + стопы со свежего боковика.
После такого upthrust падение особенно резкое: все стопы сработали, маркетмейкер уже сбросил.

**Upthrust без объёма**

Не «ненастоящий» — это показывает что покупателей осталось мало. Распределение почти завершено.
Эффект: не резкое падение а медленное сползание. Klinger начинает показывать раньше цены.

---

### SOS / SOW

**SOS через Volume Profile**

SOS через LVN = настоящий, цена летит (90%+ точность).
SOS через HVN = замедляется, может быть ложным (70% точность).
Без VP контекста SOS даёт 70%, с VP — 90%+.

**Скорость SOS = характер всего каскада**

SOS медленно 3-5 свечей на 1% = вялый каскад, откаты 40-50% от импульса.
SOS резко 1-2 свечи на 2-3% = взрывной каскад, откаты 20-30%.
Характер задаётся с первого движения и держится весь каскад.

**SOW + LPS ретест = ловушка лонгистов**

После SOW цена делает откат к уровню пробоя (LPS = Last Point of Supply):
- Объём падает (продавцов нет)
- Fisher RSI поднимается к нулю но не переходит
- Klinger немного поднимается но не пересекает ноль

Это ловушка для тех кто думает «откат и продолжение вверх». После LPS падение возобновляется.

---

### Wyckoff + система — схема решения

```
Накопление (Chop > 61.8, ADX < 20)
    ↓
Spring (wick + объём × 3, возврат за 1-2 свечи)
    → Fisher не ушёл в минус глубоко?
    → Klinger не упал ниже нуля?
    → Если оба держат = тихий spring, входи агрессивнее
    ↓
SOS (BB Width расширяется, ADX+10, Chop падает)
    → LVN выше? Сколько пространства?
    → Объём SOS > объёма spring?
    ↓
Откат к HL (Volume < импульса × 0.4)
    → Лучшая точка входа / добавления
    → Стоп: под лоем spring (не под HL)
```

# РАЗДЕЛ 4: SMC / ICT

## FVG / Order Block / Liquidity Sweep

### Концептуальная связь с системой

SMC — это Wyckoff переименованный плюс несколько новых инструментов. Главное что добавляет: **точность уровней** и **микроструктура ликвидности**.

```
Wyckoff          SMC/ICT          Volume Profile    Индикаторы
─────────────────────────────────────────────────────────────
Spring        =  SSL Sweep      =  Выход за VAL   →  Fisher, Klinger
Upthrust      =  BSL Sweep      =  Выход за VAH   →  Fisher, Klinger
SOS           =  BOS вверх      =  Пробой VAH     →  ADX+10, BB Width×2
Phase 1-2     =  Accumulation   =  POC формируется →  Chop>61.8, ADX<20
Phase 4       =  Distribution   =  POC меняет роль →  Klinger div
OB            =  HVN / POC      =  Объёмный узел  →  Точка входа на HL
FVG           =  LVN            =  Пустая зона     →  AC растёт
```

---

### FVG (Fair Value Gap / Имбаланс)

**FVG = LVN на микроуровне**

Три свечи где средняя прошла так быстро что между тенью первой и третьей осталась дыра. Механика идентична Volume Profile LVN: там не было ни продавцов ни покупателей.

**Размер FVG = энергия импульса**

- FVG 0.3% = слабый импульс
- FVG 1.5%+ = взрывной импульс, SOS максимальной энергии

**FVG не закрывается при здоровом каскаде**

При сильном движении цена идёт через несколько FVG не возвращаясь.
Незакрытые FVG = признак силы тренда.
Когда цена начинает заходить в старые FVG = первый признак затухания. Раньше чем Klinger дивергенция.

**Вложенные FVG**

FVG внутри другого FVG = двойной имбаланс. Комбинация FVG внутри LVN = максимальное ускорение.

**FVG как точка входа на откате**

Откат дошёл до верхней границы FVG и остановился + объём на заходе падает + Fisher держит бычью зону = точнее чем ATR или процент от движения.

---

### Order Block (OB)

**OB = POC на одной свече**

Последняя свеча противоположного направления перед импульсом. Там остались лимитные заявки крупняка которые не исполнились полностью. Цена возвращается = добирают остатки.

**OB + HVN = двойная поддержка**

Если Order Block находится в зоне HVN = крупняк ставил заявки там где уже был объём исторически. Такой уровень держит в 9 из 10 случаев.

**OB + HL + HVN = тройная точка входа**

Когда структурный HL + OB + HVN совпадают в 0.3% = один и тот же уровень через три метода. Вход с объёмом + Fisher подтверждает = лучшая точка во всей системе.

**Mitigation Block**

OB который уже был использован (цена дошла, отскочила, потом пробила) = становится сопротивлением. Идентично POC поменявшему роль в AMT.

**Breaker Block**

OB который не удержал цену и пробит насквозь = переворачивается в противоположную роль.
Breaker Block + CHoCH = подтверждение что разворот настоящий.

---

### Liquidity Sweep

**Иерархия ликвидности по силе**

```
Слабая:  Внутридневные хаи/лои          → движение 0.3-0.5%
Средняя: Хаи/лои последних 3-5 дней     → движение 0.7-1.5%
Сильная: Недельные / месячные хаи/лои   → движение 2-5%
```

Перед большим движением сначала sweep слабой ликвидности, потом средней. Это объясняет «ложные пробои перед настоящим движением».

**Cascade of Sweeps**

Три sweep подряд в одном направлении = методичный сбор ликвидности, не хаос.
Три sweep + Klinger монотонно растёт + Choppiness > 61.8 = взрыв неизбежен.

**BSL / SSL**

BSL (Buy Side Liquidity): стопы шортистов + тейки лонгистов = стоят ВЫШЕ хаёв. Sweep BSL = sweep вверх перед падением.
SSL (Sell Side Liquidity): стопы лонгистов + тейки шортистов = стоят НИЖЕ лоёв. Sweep SSL = sweep вниз перед ростом.

---

### Когда Liquidity Sweep ненастоящий

**Чек-лист в реальном времени:**

```
1. Объём аномальный (× 2+)?            Нет → не рассматривай
2. Ушёл за уровень на 0.3%+?           Нет → шум в пределах спреда
3. Вернулся за 1-2 свечи?              Нет → пробой, не sweep
4. Уровень имеет объёмное основание?   Нет → ненадёжно
5. Fisher / Klinger среагировали?      Нет → событие незначимое
6. За уровнем LVN / пространство?      Нет → движения не будет
```

Меньше 4 из 6 = sweep сомнительный. Пропускай.

**Признаки бредового sweep:**

- Объём на sweep равен среднему или ниже — стопы некому собирать
- Цена держится за уровнем 3+ свечи — это пробой, не sweep
- Уровень был «очевидным» слишком долго (все его видят) — там только розничные стопы, после сбора движения нет
- Sweep 0.05-0.1% — в пределах спреда, не считается
- Fisher и Klinger не среагировали — рынок проигнорировал событие
- Нет POC / HVN / OB за уровнем — нет основания для отскока

**Ретроспективная ловушка**

90% «sweepов» из обучающих материалов определены после того как движение произошло. Тест: можешь описать ПОЧЕМУ это sweep ДО того как цена пошла в нужную сторону? Если нет — это подгонка задним числом.

# РАЗДЕЛ 5: ЧТО СОЗДАЁТ РЕАЛЬНЫЙ УРОВЕНЬ

## Иерархия по надёжности

```
Кластер POC нескольких периодов + OB + HVN     ≈ 95%
Significant High/Low (объём ×2) + второй тест   ≈ 88%
Weekly Open / 52W хай/лой                        ≈ 82%
Одиночный POC / одиночный OB                    ≈ 75%
Круглые числа                                    ≈ 60%
Фибоначчи в изоляции                            ≈ 50%
Графическая фигура без объёмного основания      ≈ 45%
```

**Уровень = место где было много решений принято одновременно.** Не математика, не геометрия — психология и объём.

---

### Объёмные уровни

Прошлый POC через несколько недель/месяцев = незакрытые позиции. Когда цена возвращается — они реагируют. Это не «память цены» — это буквально незакрытые лимитные ордера.

---

### Структурные уровни

Не просто «предыдущий хай» — а хай который остановил движение на объёме выше среднего.
Там были крупные продажи → продавцы стоят там снова.

Уровень становится настоящим только после второго касания с реакцией. Первое = просто дошли. Второе + отскок на объёме = подтверждён. Третье обычно пробивает — стопы собраны.

---

### Невидимые уровни

**Цена IPO.** Там первые покупатели. При возврате выходят в безубыток.

**52-недельный хай/лой.** Там стопы всех кто держит позиции год. Sweep 52W High перед падением = один из надёжнейших сигналов.

**Цена допэмиссии / оферты.** Особенно российский рынок. Конкретные институционалы по конкретной цене. При возврате защищают или выходят.

**Экспирация опционов.** Цена тянется к страйку с максимальным открытым интересом. Не Фибо — реальные деньги маркетмейкера.

**Weekly Open.** Институционалы измеряют PnL от Weekly Open. Хеджирование создаёт реакцию на этом уровне.

---

### Алгоритм создания уровня

Вместо «нарисую Фибо»:

```
1. Найди свечу с объёмом > 2× среднего
2. Тело этой свечи = Order Block = потенциальный уровень
3. Есть ли рядом POC или HVN на Volume Profile?
4. Это значимый хай/лой структуры (остановил движение)?
5. Было ли здесь касание раньше с реакцией?
6. Если 3 из 5 = уровень настоящий
```

Сила уровня = количество независимых методов которые его видят.

---

### Ломаные графические фигуры

Классические фигуры редки в чистом виде. Правильный подход:

**Фигура = описание борьбы, не паттерн для входа.**

Треугольник:
- Покупатели делают HL (поднимают лои)
- Продавцы делают LH (опускают хаи)
- Кто-то уступит

Важно не название — а кто сжимает и кто держит.

**Вложенные фигуры = несколько таймфреймов одновременно**

Треугольник внутри флага внутри канала = не хаос, это три таймфрейма:
- Канал = недельная структура
- Флаг = дневная структура
- Треугольник = часовая структура

Все три сжимаются одновременно → прорыв одной = цепная реакция через все три.

**Ключ к ломаным фигурам:** смотри не на форму а на последовательность хаёв и лоёв внутри. Называй как угодно — если внутри HH+HL = покупатели сильнее.

# РАЗДЕЛ 6: СВЕЧНЫЕ ПАТТЕРНЫ

## Что обосновано и почему

### Главный принцип

Форма свечи без контекста = 50/50. Тот же «молот» в середине тренда = шум. На ключевом уровне после sweep = сигнал. Контекст умножает точность формы.

---

### Что реально измеряет свеча

- **Открытие/Закрытие** → кто выиграл период
- **Тени** → насколько далеко заходили проигравшие
- **Соотношение тела к теням** → степень уверенности победителя
- **Размер к соседям** → энергия события

---

### Параметры которые имеют смысл

**1. Перекрытие соседних свечей**

Свеча перекрывает N предыдущих по полному диапазону = проглотила всю их работу.
- 2-3 свечи = обычное движение
- 5-7 свечей = значимое событие
- 10+ свечей = аномалия, institutional move

Чем больше перекрытие = тем больше ордеров исполнено за этот период.

**2. Угол контекста важнее цвета**

Тень против угла движения = значимая (попытка проигравших была агрессивной).
Тень по углу движения = продолжение, не разворот.
Горизонтальный боковик + тень = просто волатильность, без смысла.

**3. Позиция закрытия внутри диапазона свечи**

```
Верхние 15%  = максимальное давление вверх
15-35%       = давление вверх
35-65%       = неопределённость (дожи)
65-85%       = давление вниз
Нижние 15%   = максимальное давление вниз
```

Это важнее чем «красная или зелёная свеча».

**4. Соотношение тела к тени**

```
Тело 80%+    = чистое направленное движение, нет борьбы
Тело 40-60%  = борьба, кто-то уступил
Тело < 20%   = равная борьба, неопределённость или разворот
Тело 0%      = идеальное равновесие (дожи)
```

**5. Объём свечи**

Любая форма умножается или делится на объём:
- Пин-бар + объём × 3 = настоящий sweep ликвидности
- Пин-бар + объём ниже среднего = случайное движение

---

### Пин-бар — правильная интерпретация

Классика: «длинная тень + маленькое тело = разворот».

Реальность: пин-бар это **событие на временном промежутке**, не форма свечи. Свеча перекрывающая 5-7 соседних = рынок прошёл недельный путь за один период. Аномалия энергии.

**Угол важнее цвета:**
Пин смотрит вниз, закрылся красным — классика говорит «слабый сигнал». Но:
- Цена ушла глубоко вниз = там были продавцы с крупными ордерами
- Цена вернулась выше открытия = покупатели поглотили всех
- Закрытие выше центра свечи = покупатели победили несмотря на цвет

**Цвет относительно открытия = мелкий шум. Позиция закрытия внутри диапазона = сигнал.**

---

### Поглощение — правильная интерпретация

**Частичное поглощение (51-80% тела)** — один период, борьба не решена.

**Полное поглощение (100-150%)** — одна сторона уничтожила работу другой.

**Каскадное поглощение** — свеча поглощает 3-5 предыдущих суммарно. Это institutional вход: объём одного периода превышает несколько предыдущих вместе.

---

### Иерархия надёжности свечного сигнала

```
Форма свечи                              → 52%
+ перекрывает 5+ соседних               → 60%
+ угол тени против тренда               → 65%
+ закрытие в верхних/нижних 15%         → 68%
+ объём × 2 и выше                      → 74%
+ на объёмном уровне (POC/HVN/OB)       → 82%
+ Fisher/Klinger подтверждают            → 88%
+ это sweep ликвидности (Equal Hi/Lo)   → 92%
```

**Свеча как аномалия, не паттерн**

Вместо поиска «молота» или «поглощения»: ищи свечу которая статистически аномальна относительно последних 20:
- Эта свеча больше среднего в X раз?
- Эта тень длиннее обычного в X раз?
- Закрытие аномально близко к экстремуму диапазона?

Аномалия = сигнал что что-то произошло. Что именно — говорит контекст (фаза, уровень, индикаторы). Сама аномалия = сигнал смотреть внимательнее, не сигнал входа.

# ИТОГОВАЯ МАТРИЦА ВСЕЙ СИСТЕМЫ

## Порядок чтения ситуации

```
1. РЕЖИМ (Choppiness, ADX)
   Боговик или тренд? Энергия накапливается или расходуется?

2. ФАЗА (Фазовый детектор)
   Накопление / Выброс / Каскад / Затухание / Разворот?

3. СТРУКТУРА (Market Structure)
   HH+HL или LH+LL? Где ближайший структурный уровень?

4. УРОВЕНЬ (Volume Profile + OB + Sweep)
   Где POC, HVN, LVN? Есть ли OB рядом? Была ли ликвидность собрана?

5. ДЕНЕЖНЫЙ ПОТОК (Klinger, OBV, Fisher)
   Подтверждает направление или дивергенция?

6. СВЕЧА/ПАТТЕРН (Price Action)
   Аномалия? Угол тени против тренда? Позиция закрытия?

7. ПАРЫ СИГНАЛОВ (матрица надёжности)
   Какие пары активны? Какая надёжность?
```

## Принцип трёх подтверждений

**Уровень**: один метод → слабый. Три метода в 0.3% → очень сильный.

**Сигнал**: одна система → 50-60%. Три независимых системы → 85-95%.

**Разворот**: CHoCH → предупреждение. CHoCH + BOS в другую сторону + Klinger ноль → настоящий.
