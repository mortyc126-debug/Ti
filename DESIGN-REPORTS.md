# Дизайн-концепция: модуль «Отчётность»

> Текущий статус: реализован как `modules/reports-full.html`, собирается из `modules/build_reports.js`.
> CSS-переменные берутся из `shell.html` / `index.html` (`:root` тема).

---

## Принципы

1. **Данные важнее навигации.** Боковая панель — инструмент быстрого поиска, не список ради списка. Активный эмитент занимает весь основной экран.
2. **Один экран — один контекст.** Нет вкладок «шапка / таблица / аудит» — всё видно сразу, по вертикальному скроллу, с якорями.
3. **Цвет = смысл.** Зелёный (рост), красный (риск), жёлтый (внимание) — строго по значению, не для декора.
4. **Компактность без потери читаемости.** Monospace для цифр, sans-serif для подписей.

## Сайдбар

### Строка инструментов (header, 36px)
- `[↓ Загрузить базу]` — accent border, uppercase, 0.6rem — единственная CTA
- `[N]` — счётчик эмитентов, monospace, text3
- `[⚙]` — 28×28px, border2, открывает `_repSourcesModal()`

### Поиск
- Полная ширина, placeholder «Поиск: эмитент, ИНН…»
- Выпадашка: эмитенты (🏢) + облигации (📄), max 6+5 строк
- Instant-search с debounce 120ms

### Список эмитентов (`.rep-issuer-item`)
Каждая строка — компактная карточка **двух строк**:

```
┌─────────────────────────────────────────┐
│ Название эмитента            год · тип  │  ← 0.65rem, жирное имя
│ ИНН xxxxxxxx  DE: 3.2x  ICR: 2.1x       │  ← 0.55rem, text3, monospace
└─────────────────────────────────────────┘
```

- Активный: `border-color: var(--acc)`, `background: var(--s3)`
- SPV/риск: `border-color: var(--warn)`
- Без данных: opacity 0.5
- **Клик** → `repSelectIssuerById(id)` — немедленно, без перерендера списка

### Фильтры (сворачиваемый блок)
Три строки пилюль:
1. **Тип:** Все · С данными · MOEX Corp · Только МСФО
2. **Отрасль:** мультиселект пилюлями (агро / металлы / …)
3. **Год:** чекбоксы 2020–2025

## Панель эмитента (main)

### Шапка
```
Название (1rem, bold)                    [✎] [🗑] [+Период] [PDF] [Сравнить]
ИНН: 1234567890  ·  Отрасль: Металлы  ·  МСФО/РСБУ
─────────────────────────────────────────────────────
Мультипликаторы:  DE: 3.2×  ND/EBITDA: 2.8×  ICR: 2.4  ROA: 8%  EBITDA%: 22%
```
Пилюли-мультипликаторы: зелёный/жёлтый/красный по порогам:
- DE < 3 → зелёный; 3–5 → жёлтый; >5 → красный
- ICR > 3 → зелёный; 1.5–3 → жёлтый; <1.5 → красный

Учредители и дочки — collapsible секция под мультипликаторами (lazy-load из D1).

### Табы периодов
```
[2024г МСФО ●]  [2023г МСФО]  [2022г РСБУ]  …  [+ Новый]
```
- Активный таб: accent underline 2px
- Тип (МСФО/РСБУ) — маленький бейдж
- ● = есть все 12 полей; ○ = неполный

### Матрица данных (эталон + сравнение)

Таблица с фиксированной первой колонкой:

| Показатель    | ед. | 2024г | 2023г | 2022г | p50 отрасли | ФНС 2023 |
|---------------|-----|------:|------:|------:|------------:|---------:|
| Выручка       | млрд| 45.2  | 38.1  | 31.4  |    —        |    —     |
| EBITDA        | млрд| 12.3  | 9.8 ▲ | 8.1   |    8.5      |    —     |
| EBITDA-маржа  | %   | 27.2  | 25.7  | 25.8  |   21.0      |  19.3    |
| …             |     |       |       |       |             |          |

- **Спарклайн** в строке (svg 60×16px) — тренд за все периоды
- Цвет ячейки: отклонение от p50 (зелёный >p75, красный <p25)
- Клик по ячейке → inline edit

### Аудит
Блок под матрицей, показывается только если есть нарушения:

```
⚠ Найдено 2 несоответствия
🔴 Активы (45.2) ≠ Капитал+Долг (48.1)  отклонение +6.4%   [→ Перейти]
🟡 Отрицательный капитал  eq = -1.2 млрд                    [→ Перейти]
```

## Цветовая семантика мультипликаторов

| Метрика      | Хорошо (зелёный) | Средне (жёлтый) | Плохо (красный) |
|--------------|-----------------|-----------------|-----------------|
| DE           | < 2.5           | 2.5–4.5         | > 4.5           |
| ND/EBITDA    | < 2.5           | 2.5–4.0         | > 4.0           |
| ICR          | > 3.0           | 1.5–3.0         | < 1.5           |
| ROA          | > 10%           | 3–10%           | < 3%            |
| EBITDA-маржа | > 25%           | 10–25%          | < 10%           |
| Current Ratio| > 1.5           | 1.0–1.5         | < 1.0           |
