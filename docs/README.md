## Глоссарий

- **Эмитент** — юридическое лицо, выпустившее облигации/акции.
- **ВДО** — высокодоходные облигации, обычно небольшие эмитенты с
  высоким yield (>15%). Часто SPV.
- **SPV** — Special Purpose Vehicle, юр. лицо специально для эмиссии
  бумаг. Часто ООО без реальной операционной деятельности; учредитель
  обычно — большая группа, чьи показатели и интересны.
- **Стресс-индекс** — композит 0..100, синтез из 6 компонентов
  (см. [E_stress.md](tracks/E_stress.md)).
- **Когорта** — группа эмитентов со схожими параметрами (отрасль,
  долговая нагрузка, размер).
- **Risk card** — структурированный отчёт о рисках эмитента,
  сгенерированный LLM на основе всех собранных данных.
- **РСБУ / МСФО** — стандарты бухучёта. РСБУ = российский, МСФО =
  международный (IFRS).
- **86-ФЗ / 402-ФЗ** — законы о бухотчётности. 402-ФЗ = обычные юр.лица
  (отчёт в ФНС, видим через ГИР БО/buxbalans). 86-ФЗ = банки (отчёт
  в ЦБ, формы 101/102).
