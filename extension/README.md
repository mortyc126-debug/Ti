# БондАналитик: сбор раскрытий с e-disclosure.ru

Расширение для Chrome / Edge / Brave. Три режима работы:

1. **Авто** — при открытии карточки компании на e-disclosure данные
   собираются автоматически и сохраняются в расширении. Ничего нажимать
   не надо.
2. **Ручной** — на карточке компании появляется синяя кнопка
   «📋 Скопировать → БА». Клик копирует JSON в буфер прямо сейчас.
3. **Массовый обход** — из popup расширения вставляешь список ИНН,
   нажимаешь «Запустить обход». Расширение поочерёдно откроет каждую
   компанию, автоматически соберёт раскрытия, закроет вкладку. Пауза
   2 сек между запросами. Можно пока заниматься другим.

После сбора — в popup кнопка **«📋 Экспорт в буфер (JSON)»**. JSON
содержит массив всех собранных компаний. В БондАналитике жмёшь
«📢 факты» у любого эмитента → в textarea вставляешь → «Импортировать».
Приложение само раскидает данные по нужным эмитентам, сверяя ИНН.

## Установка (5 минут, один раз)

### Chrome / Edge / Brave (Windows / Mac / Linux)

1. **Скачай папку `extension/`** из репозитория:
   - Открой https://github.com/mortyc126-debug/ti
   - **«Code»** → **«Download ZIP»** → распакуй
   - Найди внутри папку `extension/`
2. **Страница расширений:**
   - В адресной строке — `chrome://extensions/` (или `edge://extensions/`)
3. **Режим разработчика:**
   - В правом верхнем углу включи переключатель **«Режим разработчика»**
4. **Установи:**
   - Кнопка **«Загрузить распакованное»** → выбери папку `extension/`
5. **Готово** — в списке появится «БондАналитик: сбор раскрытий» с
   версией 1.1.0

### Firefox

Через `about:debugging#/runtime/this-firefox` → **«Load Temporary Add-on»** →
выбери `manifest.json`. Работает до перезапуска браузера. Для постоянной
установки нужна подпись или Developer Edition.

## Использование

### Режим 1: Авто (самый простой)

1. Заходи на e-disclosure.ru как обычно
2. Находи компанию → переходи на её карточку (`portal/files.aspx?id=...`)
3. Расширение автоматически собирает данные в фоне — ничего нажимать не
   нужно, счётчик в popup растёт
4. Переходи к следующей компании
5. Когда закончишь — клик по иконке расширения в правом верхнем углу
   Chrome → **«📋 Экспорт в буфер (JSON)»** → Ctrl+V в textarea
   БондАналитика

### Режим 2: Массовый обход

1. В БондАналитике нужно иметь список ИНН, которые хочешь собрать.
   Например, из портфеля или watchlist'а — скопируй их в текстовое
   поле
2. Клик по иконке расширения в правом верхнем углу Chrome
3. В поле «Массовый сбор» — вставь ИНН (по одному на строку или через
   пробелы/запятые)
4. Нажми **«🚀 Запустить обход»**
5. Браузер оставь открытым, можно свернуть. Расширение будет открывать
   вкладки одну за другой, собирать и закрывать. В popup бежит прогресс
6. Когда обход завершится — жми **«📋 Экспорт в буфер (JSON)»**
7. В БондАналитике жми «📢 факты» на любом эмитенте → вставь в
   textarea → «Импортировать»

Приложение автоматически разложит все данные по нужным эмитентам по
совпадению ИНН или имени компании.

### Режим 3: Ручной (если автосбор не сработал)

Открыл карточку компании → не вижу счётчик растёт? → нажми кнопку
«📋 Скопировать → БА» в правом верхнем углу страницы. JSON в буфер,
дальше в БондАналитик как обычно.

## Что собирается

Для каждой компании:
- Все события из таблицы существенных фактов:
  `date`, `title`, `url` (ссылка на документ)
- `companyName` — из `<h1>` или `<title>`
- `edId` — внутренний ID e-disclosure (из URL `?id=...`)
- `inn` — если найден в тексте рядом со словом «ИНН»
- `capturedAt` — timestamp сбора

## Что не делает

- Не отправляет данные в сеть — всё локально
- Не обходит anti-bot жёстко — делает 1 запрос на 2 сек, как обычный
  браузер
- Не работает вне e-disclosure.ru

## Безопасность

- `activeTab`, `tabs` — чтобы открывать/закрывать вкладки e-disclosure
- `clipboardWrite` — для экспорта JSON в буфер
- `storage` — сохранение собранных данных локально в расширении
- `host_permissions` — **только** `*://*.e-disclosure.ru/*`. Другие
  сайты расширение не видит.

Полный код — `content.js`, `background.js`, `popup.js`, `popup.html`.
Всё в открытом виде в репо, можно прочитать за 5 минут.

## Очистка данных

В popup — кнопка **«🗑 Очистить»**. Удаляет все собранные данные из
расширения (никак не влияет на то что уже импортировано в
БондАналитик).

## Обновление

Перетащи новую папку `extension/` поверх старой → на
`chrome://extensions/` нажми значок обновления ↻ рядом с расширением.

## Удаление

`chrome://extensions/` → кнопка «Удалить» рядом с расширением.
