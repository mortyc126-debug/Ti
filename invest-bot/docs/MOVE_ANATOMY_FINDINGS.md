# Анатомия хода: персистентность и развороты — рабочий журнал

Живой журнал исследования «когда движение продолжается, когда разворачивается».
Ведётся по мере проверки дыр (см. список ниже). Каждая дыра — отдельный этап:
методология → команда запуска → результаты пользователя → вывод.

Данные везде — `data/candle_cache/<TICKER>.json` (5-мин, как в score_methods).
Инструменты семейства `accel_*`.

## Что уже проверено (до этого журнала)

- **Сила ускорения × горизонт → persist%** (`accel_persistence.py`): моментум/фейд
  по силе ускорения на разных горизонтах.
- **Анатомия траектории** (`accel_excursion.py`): MFE, время до пика, откат, доля
  возврата, уход против после пика; grid [длительность × чистота × магнитуда].
- **Спайк-аномалия** (`accel_spike_test.py`): выпрыгивание ускорения из нормы =
  климакс→разворот; ускорение по/против; объём как прокси новостей; OOS-сплит.

**Итог на сейчас:** разворот определяется магнитудой (MFE) и длительностью
независимо; чистота/ER — производная; мелкие короткие спайки разворачиваются,
крупные устойчивые ходы — продолжаются (моментум).

## Этап #2 — Рыночный ход vs идиосинкразия

**Гипотеза:** движение, синхронное со всем рынком (информация), продолжается;
одиночное движение бумаги при спокойном рынке (ликвидностный шум) — фейдит.

**Метод** (`accel_breadth.py`): строим рыночный индекс-прокси = кросс-секционное
среднее m-барных доходностей по всем тикерам на каждый таймстемп. На каждом
сигнальном баре (заметный ход) классифицируем:
- **с рынком** — знак хода = знаку рынка, |рынок| ≥ медианы;
- **против рынка** — знак хода ≠ знаку рынка, |рынок| ≥ медианы;
- **идио** — |рынок| < медианы (рынок спокоен, бумага двигалась одна).

Меряем persist% и знаковый ход в ATR по горизонтам для каждой группы.

**Команда:**
```
py -3.11 accel_breadth.py --interval 5 --liquid-only
```
