# Анатомия хода: персистентность и развороты — рабочий журнал

Живой журнал исследования «когда движение продолжается, когда разворачивается».
Ведётся по мере проверки дыр (см. список ниже). Каждая дыра — отдельный этап:
методология → команда запуска → результаты пользователя → вывод.

Данные везде — `data/candle_cache/<TICKER>.json` (5-мин, как в score_methods).
Инструменты семейства `accel_*`.

## Что уже проверено (до этого журнала)

- **Сила ускорения × горизонт → persist%** (`accel_persistence.py`): моментум/фейд
  по силе ускорения на разных горизонтах.
- **Анатомия траектории** (`accel_excursion.py`): MFE, время до пика, откат, доля
  возврата, уход против после пика; grid [длительность × чистота × магнитуда].
- **Спайк-аномалия** (`accel_spike_test.py`): выпрыгивание ускорения из нормы =
  климакс→разворот; ускорение по/против; объём как прокси новостей; OOS-сплит.

**Итог на сейчас:** разворот определяется магнитудой (MFE) и длительностью
независимо; чистота/ER — производная; мелкие короткие спайки разворачиваются,
крупные устойчивые ходы — продолжаются (моментум).

## Стратегии-кандидаты (честный бэктест фейда)

`fade_backtest.py` — no-overlap бэктест фейда резкого хода (вход ПРОТИВ), та же
строгая механика, что nw_backtest (интрабар тейк/стоп стоп-первым, cost, OOS,
разбивка long/short). Фильтры из находок:
- `--filter breadth` — фейдим идио/против-рынка ход (#2);
- `--filter level`   — фейдим ход, упёршийся в прошлый экстремум (#6);
- `--filter both`.

**Результаты** (171 тикер, take0.5/stop0.75, cost0.05, maxhold12, exp ATR/сделку):
```
фильтр    TRAIN exp   TEST exp   TEST long(фейд↓пад)  TEST short(фейд↑рост)  TEST N
breadth   +0.090      +0.018     -0.007               +0.042                 169387
level     +0.091      -0.002     -0.035               +0.043                  18085
both      +0.186      +0.065     +0.042               +0.092                   9448
```

**Вывод:**
1. **Фейд-эдж реален и переживает OOS — но хрупкий.** Сильный распад TRAIN→TEST
   во всех фильтрах (both 0.186→0.065, ~65% эджа испарилось). NW проседал куда
   меньше (0.092→0.083) → фейд заметно РЕЖИМНЕЕ NW.
2. **`both` (breadth ∧ level) — лучший OOS: +0.065 ATR, win 69%, обе стороны в
   плюсе.** Стакание фильтров = чистейший сетап (но N=9448, поток тонкий).
3. **Шорт-фейд (фейд роста) везде сильнее** и держит OOS (+0.04…+0.09); лонг-фейд
   (фейд падений) мёртв/минус в breadth/level. Прямо подтверждает #4: ралли
   откатывают, распродажи тянутся.
4. **ОГОВОРКА (важно):** OOS-выживание сосредоточено в шорт-фейде, а он частично =
   медвежья бета TEST-окна (рынок падал → ралли гасли, падения тянулись). Часть
   OOS-плюса — тот же дрейф, что в NW-беты-разложении и #4. Не факт, что структурно.
5. Win 65-79% высок механически (тейк 0.5 ближе стопа 0.75), exp тонкий из-за
   асимметрии брекета + cost. Тейк/стоп под свип (откат ~0.25 ATR — тейк можно уже).
6. **Итог:** жизнеспособный, но хрупкий шорт-смещённый mean-reversion оверлей.
   `both` — рабочая точка. NW надёжнее; фейд — второй поток, не замена. Перед live —
   свип тейк/стоп и шорт-онли; относиться как к режимному, не сайзить агрессивно.

## Общий синтез (все 6 этапов замкнуты)

Всё сходится в одну картину «когда фейд, когда моментум»:

- **Фейд (разворот) сильнее, когда ход = шум/локальная аномалия:** идиосинкразия
  (#2), тонкие часы (#1), упор в прошлый уровень/реджект (#6), спайк из тихой базы
  (#5). Такой ход возвращается.
- **Держится (моментум), когда ход = информация/поток:** синхронно с рынком (#2),
  ядро ликвидной сессии (#1), пробой уровня (#6).
- **Сильная асимметрия направления (#4):** ралли фейдят (тем сильнее, чем крупнее),
  распродажи на длинном горизонте продолжаются. Рационализирует шорт-онли NW-эдж —
  но частично это медвежья бета периода (осторожно, не структурная гарантия).
- **Squeeze-брейкаут НЕ подтвердился (#5)** — на 5-мин ход из сжатия скорее фейдит.
- **Режим (#3) — слабый модулятор:** фейд держится во всех режимах (Симпсон снят),
  боковик даёт чуть сильнейший откат; не переключатель момент/фейд.

Практический дистиллят для фейд-логики: фейдить резкий ход, если он
идио/против-рынка (#2) И упирается в уровень (#6) И вне ядра сессии учитывая
ликвидность (#1) И это РОСТ (#4). Не фейдить: рыночный ход, пробой, распродажу
на удержании.

## Этап #2 — Рыночный ход vs идиосинкразия

**Гипотеза:** движение, синхронное со всем рынком (информация), продолжается;
одиночное движение бумаги при спокойном рынке (ликвидностный шум) — фейдит.

**Метод** (`accel_breadth.py`): строим рыночный индекс-прокси = кросс-секционное
среднее m-барных доходностей по всем тикерам на каждый таймстемп. На каждом
сигнальном баре (заметный ход) классифицируем:
- **с рынком** — знак хода = знаку рынка, |рынок| ≥ медианы;
- **против рынка** — знак хода ≠ знаку рынка, |рынок| ≥ медианы;
- **идио** — |рынок| < медианы (рынок спокоен, бумага двигалась одна).

Меряем persist% и знаковый ход в ATR по горизонтам для каждой группы.

**Команда:**
```
py -3.11 accel_breadth.py --interval 5 --liquid-only
```

**Результаты** (143 тикера, индекс 198k таймстемпов, медиана |M|=0.00041,
ход = |Δ3бар| ≥ 0.5 ATR):

persist% (>50 продолжил, <50 развернулся):
```
группа              n         5м    10м    30м    60м   240м
идио (рынок тих)  1011568   32.1   34.2   37.1   39.0   43.0
С рынком          1109945   41.4   42.6   44.9   45.7   47.9
против рынка       196721   38.2   40.3   42.0   43.1   45.3
```
знаковый ход в ATR (>0 продолжил, <0 развернулся):
```
группа              n         5м    10м    30м    60м   240м
идио              1011568  -0.223 -0.244 -0.277 -0.297 -0.319
С рынком          1109945  -0.053 -0.060 -0.044 -0.047 -0.007
против рынка       196721  -0.183 -0.202 -0.250 -0.279 -0.354
```

**Вывод (гипотеза #2 подтверждена сильно):**
1. **Все группы разворачиваются** (persist<50, знак<0) — потому что сигнал =
   резкий ход ≥0.5 ATR за 3 бара, а резкие пуши в среднем откатывают (совпадает
   с прошлым: короткие спайки фейдят). Это фон — mean-reversion после пуша.
2. **Breadth чисто сортирует силу отката, монотонно по всем горизонтам:**
   - **идио** (одиночный ход, рынок тих) — откатывает СИЛЬНЕЕ всех: −0.22…−0.32 ATR.
     Ликвидностный шум возвращается. Это лучший фейд.
   - **С рынком** — почти НЕ откатывает (−0.05…−0.01): рыночный ход держится.
     Фейдить нельзя — это моментум.
   - **против рынка** — откатывает почти как идио (−0.18…−0.35): ход против
     рынка утягивает назад, на длинном горизонте разворот сильнейший (−0.354).
3. **Практика:** резкий ход стоит фейдить ТОЛЬКО если он идиосинкразический или
   против рынка; синхронный с рынком — не трогать. Магнитуда отката идио (−0.22…
   −0.32 ATR) КРУПНЕЕ NW-эджа (0.05) — breadth-фильтрованный фейд заслуживает
   отдельной проверки как самостоятельная стратегия (этап-кандидат).
4. Для NW-шорта: осторожно — тут другой сигнал (пост-пуш фейд), прямой перенос
   некорректен, но связь есть — breadth как фильтр направления перспективен.

_Оговорка: выборка перекрывающаяся (не no-overlap), поэтому абсолютная магнитуда
−0.22 не равна доходности сделки; но относительный порядок групп чист._

## Этап #1 — Время суток / фаза сессии

**Гипотеза:** на MOEX жёсткая внутридневная структура (утренний импульс, всплеск
на открытии США ~13:30 UTC / 16:30 MSK, предзакрытие, вечёрка) → склонность хода
продолжаться/разворачиваться зависит от времени. По времени раньше не резали.

**Метод** (`accel_session.py`): тот же сигнал (резкий ход ≥ move-atr ATR), но
группируем по часу таймстемпа (свечи в UTC; печатаем UTC и MSK=+3). Persist% и
знаковый ход в ATR по горизонтам для каждого часа.

**Команда:**
```
py -3.11 accel_session.py --interval 5 --liquid-only
```

**Результаты** (143 тикера, ход ≥0.5 ATR). Знаковый ход в ATR по часам (U=UTC,
M=MSK+3), сжато:
```
час(MSK)   n        5м     30м     60м    persist@60м
03-06    ~11-20k  -0.44…-0.47 -0.53 -0.52…-0.58   22-32%   тонкая ночь
07-09    ~90-125k -0.18…-0.22 -0.24 -0.21…-0.29   39-45%   утро, откат слабеет
10-17    ~145-170k -0.05…-0.11 -0.04…-0.11 ≈-0.02…-0.14  38-45%  ЯДРО, держится
17:00    149k     -0.086 -0.040 -0.019            45%     «залипает» лучше всех
18-19    ~127-130k -0.15…-0.18 -0.21…-0.24 -0.22…-0.28  39-42%  вечер, откат растёт
22-02    ~17-18k  -0.64 -0.78 -0.83               24-27%   поздняя вечёрка, тонко
```

**Вывод (гипотеза #1 подтверждена):**
1. **Сильная внутридневная сезонность отката, U-образная.** Откат меняется в
   5-10× по времени суток.
2. **Ядро ликвидной сессии (10-17 MSK) — минимальный откат** (−0.05…−0.11):
   ходы держатся лучше всего, момент-дружелюбное окно. Пик «залипания» ~17:00 MSK.
3. **Тонкие часы (ночь, поздняя вечёрка) — жесточайший фейд** (−0.44…−0.83), но
   N в 5-10× меньше и это малоликвид: откат отчасти артефакт спреда (одиночный
   принт отскакивает), торговать нельзя без учёта широкого спреда.
4. **Вечер после 18 MSK — откат снова растёт** (−0.15…−0.28): смена характера
   после закрытия основной сессии.
5. Практика: фейд-стратегии резких ходов работают вне ядра (утро/вечер), но с
   поправкой на ликвидность; в ядре моменты держатся — фейдить осторожнее. Для
   NW-шорта (ликвид, ядро) время благоприятно: в ядре нет сильного отскока вверх.

## Этап #3 — Режим (тренд/боковик)

**Гипотеза:** в тренде ход ПО тренду продолжается, ПРОТИВ — откатывает; в боковике
всё откатывает. Персистентность условна по режиму (Симпсон-риск).

**Метод** (`accel_regime.py`): режим = Efficiency Ratio Кауфмана на окне W
(ER≥порога = тренд). Группы: ТРЕНД по / ТРЕНД против / БОКОВИК.

**Команда:**
```
py -3.11 accel_regime.py --interval 5 --liquid-only
```

**Результаты** (143 тикера). ER≥0.5 на 60 барах оказался ОЧЕНЬ строг: почти всё
(2.31М) попало в «боковик», в тренд — крохи (2796 по / 513 против).
```
режим          n         5м     30м     60м    240м
ТРЕНД по      2796    -0.137 -0.002 +0.203 -0.009   (шумно, мало N)
ТРЕНД против   513    +0.011 +0.119 +0.267 +1.055   (шум, N=513)
БОКОВИК     2310881   -0.138 -0.163 -0.176 -0.173   (= базовый фейд)
```

Первый прогон (ER≥0.5) голодал (99.9% боковик). **Перезапуск с ER≥0.3** наполнил
вёдра:
```
режим          n         5м     30м     60м    240м    persist@60м
ТРЕНД по      78364   -0.076  -0.137  -0.157  -0.070    42.4%
ТРЕНД против  24710   -0.014  +0.021  +0.014  -0.082    50.4%
БОКОВИК     2211116   -0.141  -0.166  -0.178  -0.177    42.5%
```

**Вывод (гипотеза #3 в наивном виде НЕ подтверждена; режим — слабый модулятор):**
1. **Во всех режимах доминирует фейд спайка** (знак ≤0 везде) — нет скрытого
   моментум-подрежима, где знак переворачивается. **Страх Симпсона снят.**
2. Режим модулирует СИЛУ отката:
   - **боковик** — сильнейший фейд (−0.14…−0.18): диапазонная реверсия, ожидаемо;
   - **тренд, ход ПО тренду** — фейд слабее (−0.08…−0.16): спайк перелетает тренд
     и подтягивается назад, но трендовая компонента держит;
   - **тренд, ход ПРОТИВ** — почти нейтр. (~0, persist ≈50%): контртрендовый спайк
     не откатывает и не продолжается (баланс).
3. «Моментум только по тренду» — НЕ подтвердилось: даже ход по тренду в среднем
   слегка фейдит. Практика: режим — второстепенный фильтр (боковик = чуть лучший
   фейд), не переключатель момент/фейд.

## Этап #4 — Асимметрия вверх/вниз

**Гипотеза:** падения острее ростов → развороты после ходов вверх/вниз разные.
Прямо релевантно шорт-онли NW.

**Метод** (`accel_asym.py`): разрез по знаку хода (вверх/вниз) × полосам магнитуды
(в ATR). Сравнение зеркальных ячеек.

**Команда:**
```
py -3.11 accel_asym.py --interval 5 --liquid-only
```

**Результаты** (143 тикера). Знаковый ход в ATR (>0 продолжил, <0 развернулся):
```
ВВЕРХ    магнит      5м      30м     60м    240м
         0.5-1    -0.082  -0.117  -0.147  -0.273
         1-1.5    -0.143  -0.188  -0.220  -0.392
         1.5-2.5  -0.170  -0.227  -0.279  -0.532
         >2.5     -0.264  -0.288  -0.342  -0.682
ВНИЗ     0.5-1    -0.079  -0.085  -0.080  +0.057
         1-1.5    -0.162  -0.178  -0.171  +0.006
         1.5-2.5  -0.198  -0.215  -0.200  +0.050
         >2.5     -0.270  -0.249  -0.195  +0.090
```

**Вывод (САМЫЙ ВАЖНЫЙ этап — сильная асимметрия):**
1. **Ходы ВВЕРХ разворачиваются, и тем сильнее, чем крупнее и дольше:** крупный
   рост (>2.5 ATR) к 240м отдаёт −0.68 ATR. Ралли фейдят.
2. **Ходы ВНИЗ короткосрочно отскакивают, но к 240м ПРОДОЛЖАЮТСЯ** (знак
   становится ПОЛОЖИТЕЛЬНЫМ +0.05…+0.09): падение не разворачивается на длинном
   горизонте, а тянется дальше вниз.
3. **Структурная асимметрия рынка (в этих данных): ралли откатывают, распродажи
   продолжаются.** Это ПРЯМО объясняет, почему NW-эдж шорт-онли: нисходящие ходы
   держатся, восходящие фейдят.
4. **ВАЖНАЯ ОГОВОРКА:** это может быть бета периода — в окне рынок в среднем падал
   (см. #2: null-шорт +0.037), поэтому «низ продолжается» отчасти = медвежий дрейф.
   Не факт, что структурно навсегда. Тот же риск, что в NW-разложении беты.
5. Практика: шорт резкого ПАДЕНИЯ рискован коротко (отскок −0.27 на 5м даже вниз),
   но оправдан на удержании (к 240м +0.09). Фейд резкого РОСТА — надёжнее и растёт
   с горизонтом. Асимметрию стоит учесть в тейк/стоп и горизонте.

## Этап #5 — Сжатие → расширение (squeeze)

**Гипотеза:** ход из низковолатильной базы (сжатие) = прорыв, продолжается; ход в
уже-высокой воле = выхлоп, разворачивается.

**Метод** (`accel_squeeze.py`): состояние воли на баре ПЕРЕД ходом (i−m) =
ATR / медиана ATR за W баров. Сжатие / норма / расширение.

**Команда:**
```
py -3.11 accel_squeeze.py --interval 5 --liquid-only
```

**Результаты** (143 тикера). Знаковый ход в ATR по состоянию воли ДО хода:
```
состояние        n         5м     30м     60м    240м
СЖАТИЕ (тихо)  686528   -0.171  -0.200  -0.210  -0.246
норма         1117428   -0.129  -0.153  -0.167  -0.152
РАСШИР(шумно)  495950   -0.112  -0.135  -0.149  -0.120
```

**Вывод (гипотеза #5 ОПРОВЕРГНУТА — знак обратный):**
1. Ожидали: ход из сжатия (прорыв) держится, ход в высокой воле (выхлоп) фейдит.
   **Данные — наоборот:** ход из СЖАТИЯ откатывает СИЛЬНЕЕ (−0.25), ход в
   РАСШИРЕНИИ — слабее (−0.12). Монотонно, N огромные — эффект реальный, но умеренный.
2. Причина: резкий 0.5+ ATR ход из тихой базы — это спайк в вакуум (низкое участие),
   он отскакивает; ход в уже-высокой воле — часть идущего движения, продолжается
   относительно больше. Плюс нормировка: 0.5 ATR при низком ATR = мелкий абсолютный
   ход, чаще шум.
3. Практика: «пробой из сжатия» на 5-мин как сигнал ПРОДОЛЖЕНИЯ не подтвердился —
   скорее фейд-сетап. Squeeze-брейкаут в чистом виде тут не работает.

## Этап #6 — Контекст уровня

**Гипотеза:** ход В прошлый экстремум отбивается (реджект→разворот); ход, пробивший
экстремум, продолжается или ложно-разворачивается; в свободном поле — нейтр.

**Метод** (`accel_level.py`): прошлый диапазон [Lmin,Hmax] за N баров до хода.
Класс по положению close и знаку: ПРОБОЙ / В УРОВЕНЬ / СВОБОДНО.

**Команда:**
```
py -3.11 accel_level.py --interval 5 --liquid-only
```

**Результаты** (143 тикера). Знаковый ход в ATR + persist@60м:
```
класс         n         5м     30м     60м    240м   persist@60м
ПРОБОЙ     221124   -0.123  -0.133  -0.149  -0.112     42.5%
В УРОВЕНЬ  115770   -0.208  -0.233  -0.259  -0.176     37.8%  ← сильнейший фейд
СВОБОДНО  1973085   -0.135  -0.163  -0.174  -0.180     42.8%
```

**Вывод (гипотеза #6 подтверждена):**
1. **Ход В прошлый экстремум (к сопротивлению/поддержке) разворачивается сильнее
   всех** (−0.21…−0.26, persist 33-42%) — реджект от уровня. Лучший фейд-сетап.
2. **Пробой прошлого экстремума откатывает МЕНЬШЕ** (−0.11…−0.15) — брейкаут
   держится относительно лучше (хотя в среднем всё равно чуть в минус: ложные
   пробои есть, но пробой лучше базы).
3. «Свободно» — базовый фон.
4. Практика: уровень — реальный модификатор. Фейдить ход, упирающийся в прошлый
   хай/лоу (реджект); не фейдить пробой. Совмещается с фильтрами #1/#2.
